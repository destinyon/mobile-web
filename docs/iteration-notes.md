# 迭代说明

## 第 1 轮

- 补齐仓库基础说明与忽略规则。
- 新建 Spring Boot 后端骨架、H2 数据库、初始化数据与核心 API。
- 新建原生微信小程序页面骨架、组件、请求封装和主题样式。

## 第 2 轮

- 完成新闻列表、详情、收藏、点赞、评论、我的收藏、我的评论、我的发布、发布内容与后台接口。
- 补充微信登录、OSS 上传、接口文档和集成测试。
- 将后端搜索查询改为参数化，避免字符串拼接。
- 在微信开发者工具中抓取首页、我的页、新闻详情页截图，确认小程序可加载真实后端数据。

## 第 3 轮

- 修复微信登录 500：拆分 `WxSessionClient`，将微信错误码和网络异常转换为业务响应。
- 修复种子数据自增序列未推进导致首次登录插入用户撞主键的问题。
- 接入聚合新闻同步：新增 Juhe 配置、客户端、同步服务和 `POST /api/admin/news/sync/juhe` 后台接口。
- 在 `.env` 中加入聚合新闻 AppKey、接口地址、体育分类、分页配置和 50 个羽毛球关键词。
- 重写中文乱码文件，修复搜索页 placeholder 属性错误。
- 优化小程序 UI：更贴近原型的浅绿底、黑描边、紧凑新闻卡片、详情页标题栏、个人中心列表式功能区。
- 验证结果：后端 `mvn test -q` 通过；小程序 JS 语法和 JSON 解析检查通过。
